## SDG Research Pipeline

This repository contains a research pipeline for multi-label classification of UN Sustainable Development Goals (SDGs). It is organized to support:

- Reproducible baselines from existing benchmark work.
- Synthetic SDG data generation and validation using LLMs.
- Modern embedding-based classifiers and cross-dataset experiments.

For now, the focus is on:

- Loading and analyzing the SDGi corpus.
- Computing detailed dataset statistics needed for downstream modeling and evaluation.
